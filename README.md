# SmartSoft AI GPU Optimization

## Описание проекта

Данный проект разработан в рамках производственной практики и адаптирован под деятельность ООО «СМАРТ-СОФТ».

ООО «СМАРТ-СОФТ» занимается разработкой программного обеспечения, созданием сайтов, интернет-магазинов, мобильных приложений, интеграцией 1С и автоматизацией бизнес-процессов. В связи с этим задача оптимизации программ, использующих графический процессор для ускорения искусственного интеллекта, была адаптирована под eCommerce-направление.

Проект представляет собой AI-модуль для автоматической классификации изображений товаров интернет-магазина. Программа использует предобученную нейронную сеть MobileNetV2 и автоматически выбирает устройство обработки: GPU при наличии видеокарты с поддержкой CUDA или CPU при отсутствии GPU.

## Цель проекта

Цель проекта — повысить производительность обработки изображений товаров за счет использования графического процессора.

## Функциональность

Программа выполняет следующие действия:

1. Загружает изображения из папки `images`.
2. Проверяет доступность GPU.
3. Загружает предобученную модель MobileNetV2.
4. Обрабатывает изображения с помощью нейронной сети.
5. Сохраняет результаты классификации в файл `results.csv`.
6. Выводит время обработки и используемое устройство.

## Используемые технологии

- Python
- PyTorch
- Torchvision
- MobileNetV2
- Pillow
- Pandas
- CUDA / CPU
